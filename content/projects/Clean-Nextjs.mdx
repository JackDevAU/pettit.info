**Aim:** Quickly be able to get spun up with an opinionated and "clean" approach to using Next.js. 

**Features:**

* Init Command
  * Create a Clean Next.js project in a single command!
* Add Command
  * Add packages to your project, like biomejs, shadcnui etc.
* Create Command
  * Quickly create pages and components
