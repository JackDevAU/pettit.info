**Project Overview**

A Next.js implementation of a budgeting app, inspired by [UpBlazor](https://upblazor.com/), designed to interact with the Up Bank API for effortless financial tracking.

**Features:**

* Track Incomes
  * Create Income (Weekly, Fortnightly, Monthly, Quarterly, Yearly)
* Track Expenses
  * View and Create & Remove Tags
* Insights and analytics
  * Total Expenses

Project Goals

* Provide a user-friendly interface for managing finances
* Offer insights into spending habits and income
