**Aim:** A Next.js implementation of an Up Bank Budgeting app interacting with the Up Bank Api inspired by [UpBlazor](https://github.com/Hona/UpBlazor) for keeping track of finances.

**Features:**

* Track Incomes
  * Create Income (Weekly, Fortnightly, Monthly, Quarterly, Yearly)
* Track Expenses
  * View and Create & Remove Tags
* Insights and analytics
  * Total Expenses
